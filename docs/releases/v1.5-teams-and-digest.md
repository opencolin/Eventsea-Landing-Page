# v1.5 — Team workspaces + saved views + weekly digest

> Status: **planned**.
> Branch: `release/v1.5-teams-and-digest`
> Estimated effort: 4–5 weeks for 2 engineers
> Depends on: v1.0 with ≥10 paying customers

## Goal

Make Eventsea sticky inside teams. The product so far is single-user. v1.5 ships team workspaces, shared filter sets, per-persona weekly digests, and Slack integration so the product earns a recurring slot in someone's Monday morning.

Success metric: **≥40% of active orgs have ≥2 active users in the past 14 days. ≥30% of users open the weekly digest each week.**

## Non-goals

- Not building the sponsor marketplace yet (v2.0).
- Not building advanced analytics or dashboards. Stay focused on the recurring read.

## In-scope features

### 1. Team workspaces
- Org owners can invite teammates with `member` role.
- Shared saved views: `vcluster GTM`, `Platform Leads`, `AI Infra Builders` (per the screenshot).
- Per-view filter weights and custom filters are shared. Per-user notes + P1/P2/P3 remain private by default; toggleable per view.

### 2. Per-persona weekly digest
- Each user picks a saved view as their digest source.
- Monday 7am local time: digest email with the top 10 events ranked, with note prompts.
- Clickable to mark P1/P2/P3 from email.
- Implementation: Resend + scheduled job.

### 3. Slack app
- One-time per-org install.
- `/eventsea radar` slash command returns the top 5 events of the user's saved view.
- New event matches above a threshold post to a configured channel.
- Implementation: lightweight Slack app, just slash command + webhook posts.

### 4. Audit log + collaboration signal
- Each user's P1/P2/P3 + notes are visible to teammates in the shared view.
- Notification badge: "3 teammates marked these P1 this week."

## Implementation checklist

- [ ] Saved views: convert to shareable resources with ACLs

## Acceptance test

A team of 4 signs up. Each member chooses a saved view. On Monday morning, each member gets a digest. By Thursday, the team has marked 30+ events as P1/P2/P3 collaboratively. Slack channel shows ~5 high-fit events posted automatically that week.
