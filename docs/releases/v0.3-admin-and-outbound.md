# v0.3 — Admin + outbound + first 5 design partners

> Status: **planned**.
> Branch: `release/v0.3-admin-and-outbound`
> Estimated effort: 6–8 eng-days + ongoing founder GTM time
> Depends on: v0.2 shipped

## Goal

v0.2 made the wedge real. v0.3 turns wedge submissions into pipeline. By end of v0.3, **5 design partners are using the audit + radar regularly** — at least 3 from the named target list (Nebius, vCluster, Tavily, LanceDB).

Success metric: **5 design partner agreements signed, ≥3 from the named target list, ≥2 paying.**

## Non-goals

- Not building a full CRM. Notion / Airtable + the admin page is enough.
- Not building self-serve org signup. Design partners are hand-onboarded.
- Not the team digest yet (v1.5).

## In-scope features

### 1. Admin dashboard
- `/admin` page, password-gated (single shared password env var for now).
- Lists: calendar audits (with status), marketplace listings, beta signups, demo requests.
- Per-audit detail view: scraped events, generated report, email status, manual "re-run" button.
- Implementation: `client/src/pages/admin.tsx` + simple JWT middleware.

### 2. Audit-as-cold-outreach machine
- Founder pastes a target's Luma calendar into the admin form
- Eventsea generates an audit, but instead of emailing the calendar owner, generates a **personalized outbound email** with the headline insight ("your AI Infra Night was your #1 event — here's how to do it 3x more often")
- Founder copies the email into HubSpot or just sends manually
- Implementation: `server/audit/outreach-email.ts` — new Claude prompt variant that frames findings as an outreach hook

### 3. Public sample reports for SEO + social
- Generate audits for 20 well-known calendars (Y Combinator, ETHDenver, ETHGlobal, Replit, Latent Space, AI Engineer Summit, etc.)
- Host them publicly at `/audits/:slug`
- Each one is a SEO landing page for that calendar's organizer + audience
- Implementation: a CLI script `server/scripts/seed-public-audits.ts` that drops them into the DB

### 4. Outbound sequence templates
- Doc only (no code): `docs/gtm/outbound-sequences.md` with copy for:
  - DevRel at AI infra companies (Nebius, Vercel, Replicate, Modal, Tavily)
  - Field marketing at enterprise SaaS (Snowflake, Datadog, Vercel)
  - Conference organizers (Y Combinator, Latent Space, AI Engineer)
- Each sequence: cold + 2 follow-ups + an "audit" hook variant

### 5. Calendly / SavvyCal integration
- Replace the `mailto:` Book-a-Demo with a real Calendly embed.
- Add a 15-min audit-readout call slot type.
